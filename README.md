# DesktopPony v1.0.1-beta

-基于PT素材的开源桌宠，目前exe版仅支持1980x1080的64位windows系统，Python版记得安装相应的库
os
sys
random
PyQt5
cv2(OpenCV2)
numpy
PIL
win32gui	# 仅限Win
shutil

-目前只有一个实用功能：触发试备忘录(一开始写桌宠就是为了这么一个功能x)(详见文件内说明)

-下次更新会加入更多分辨率选项和一些新功能

-未来会加入更多动作和自定义OC，当然这个要等研究完PT源代码和素材的组织形式

（遇到bug请贴在评论区，还没有时间搭虚拟机测试在不同系统下的稳定性，我自己的64位win10和32位win7都能跑exe版）
